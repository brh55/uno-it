# uno-it [![Build Status](https://travis-ci.org/brh55/uno-it.svg?branch=master)](https://travis-ci.org/brh55/uno-it)

> Sum a collection of units into one single unit


## Install

```
$ npm install --save uno-it
```


## Usage

```js
const unoIt = require('uno-it');

unoIt('unicorns');
//=> 'unicorns & rainbows'
```
